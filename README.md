# JS Counter DOM

A lightweight JavaScript counter application that demonstrates basic DOM manipulation and event handling.

## Features
- Increase the counter value
- Decrease the counter value
- Reset the counter to zero
- Real-time UI updates using JavaScript

## Tech Stack
- HTML
- CSS (optional styling)
- JavaScript (Vanilla)

## How It Works
The application keeps a single `count` variable in memory.  
Button clicks trigger event listeners that update the value and reflect changes in the DOM using `textContent`.

## Core Logic
- `increase()` → increments count by 1
- `decrease()` → decrements count by 1
- `countReset()` → resets count to 0

## Learning Goals
This project helps reinforce:
- DOM selection (`getElementById`)
- Event listeners (`addEventListener`)
- State management in vanilla JS
- Basic UI updates

## Setup
1. Clone the repository
2. Open `index.html` in a browser
3. Click buttons to interact with the counter
